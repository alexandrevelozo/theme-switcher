<h2 id="running-locally"> Running locally </h2>

From your command line:

```sh
# download the repository
> git clone https://github.com/alexandrevelozo/theme-switcher

# server
> yarn install
> yarn start

# web
> yarn install
> yarn start

# mobile
> yarn install
> yarn start
```
